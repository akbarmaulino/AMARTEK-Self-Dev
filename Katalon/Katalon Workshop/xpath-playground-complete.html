What is XPath? ▼
XPath adalah bahasa query untuk navigasi element dalam dokumen XML/HTML.
Kenapa pakai XPath di Katalon? ▼
Karena XPath bisa navigasi berdasarkan struktur dan relasi antar element, tidak hanya attribute.

Element di dalam iframe punya DOM sendiri. Di Katalon harus switchToFrame() dulu sebelum bisa akses element-nya.

// 1. Switch ke iframe dulu

// 2. Baru akses element di dalamnya

// 3. Selesai, balik ke parent
WebUI.switchToDefaultContent()
